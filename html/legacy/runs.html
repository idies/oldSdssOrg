The Legacy Survey: The Runs Database

The DR7 Catalog Archive Server contains the imaging RUNS database, separate from the DR7 (Legacy + SEGUE) database. This database contains 530 imaging scans, some of which are not present in the main DR7 database, and thousands of imaging fields from the beginnings and ends of DR7 strip scans which did not resolve into a complete rectangular stripe on the sky.
