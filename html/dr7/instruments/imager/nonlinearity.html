Imager CCD nonlinearity

We designate CCDs using the filter (which corresponds to the row) and column. For example, the g4 CCD is the CCD in the fourth column in the row covered by the g filter. The nonlinearity plots below are grouped by filter. In these plots the filled points belong to amplifier 0, the empty ones to amplifier 1 (if applicable). The Amplifier 1 fits are dashed. The vertical dotted line is the adopted saturation level for that device; points to the right of this line are ignored in the fits.

The fits shown were to the function::

Signal = A × Count × ( 1 + α × f ( Count / ct ) )

where α and ct are fitted quantities, and
f ( x ) = 0 for x < 1
f ( x ) = log10(x) for x > 1.

This function represents the nonlinearity in these CCDs well.
