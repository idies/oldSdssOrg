SDSS pipelines

The pipelines

Each of the operating steps is actually a collection of algorithms run in pipelines producing a number of data products. Here, we give a brief description of what is contained in these steps, together with links to more detailed explanations. Again, refer to descriptions of the Catalog Archive Server contents and the Data Archive Server contents as well as details of our algorithms elsewhere.

Photometric Telescope (PT) / Monitoring Telescope (MT) pipeline

The photometric telescope (PT) has two main functions: it measures the atmospheric extinction on each clear night based on observations of primary standards at a variety of air masses, and it calibrates secondary patches in order to determine the photometric zero point of the 2.5 m imaging scans. The monitor telescope pipeline (mtpipe), so called for historical reasons, processes the PT data.

Imaging pipelines

The imaging pipelines include the following:

  • the astrometric pipeline astrom, which performs the astrometric calibration
  • the photo pipeline, consisting of two components:
    1. the postage stamp pipeline psp, which characterizes the behavior of the point-spread function as a function of time and location in the focal plane
    2. the frames pipeline, which finds, deblends, and measures the properties of objects
  • the final calibration pipeline nfcalib which applies the photometric calibration to the objects
