SDSS Data Archive Server Quick Reference

Most Commonly Useful Files

Imaging

File Type Description DAS URL format format parameters
fpC the image itself, bias subtracted, flat fielded, and with bad pixels replaced by interpolated values (the "corrected frame") http://das.sdss.org/imaging/%d/%d/corr/fpC-%06d-%c%d-%04d.fit run, rerun, run, filter, camcol, field
drObj the final "ubercal" and astrometrically recalibrated object catalogs http://das.sdss.org/imaging/%d/%d/dr/%d/drObj-%06d-%d-%d-%04d.fit run, rerun, camcol, run, camcol, rerun, field
drField the final "ubercal" field statistics http://das.sdss.org/imaging/%d/%d/dr/%d/drField-%06d-%d-%d-%04d.fit run, rerun, camcol, run, camcol, rerun, field
tsObj the calibrated object catalogs (as used for targeting) http://das.sdss.org/imaging/%d/%d/calibChunks/%d/tsObj-%06d-%d-%d-%04d.fit run, rerun, camcol, run, camcol, rerun, field
tsField the calibrated field statistics (as used for targeting) http://das.sdss.org/imaging/%d/%d/calibChunks/%d/tsField-%06d-%d-%d-%04d.fit run, rerun, camcol, run, camcol, rerun, field
asTrans tranformation coefficients from row, column to great circle coordinates http://das.sdss.org/imaging/%d/%d/astrom/asTrans-%06d.fit run, rerun, run
fpBIN a 4x4 binned version of the corrected image after masking of objects and subtraction of sky, an image of the estimate of the sky, an image of the uncertainty in the sky estimate, and a table of bright star wings http://das.sdss.org/imaging/%d/%d/objcs/fpBIN-%06d-%c%d-%04d.fit run, rerun, run, filter, camcol, field
fpM the frame masks (read using readAtlasImages) http://das.sdss.org/imaging/%d/%d/objcs/fpM-%06d-%c%d-%04d.fit run, rerun, run, filter, camcol, field
fpAtlas the atlas images for all objects detected (read using readAtlasImages) http://das.sdss.org/imaging/%d/%d/objcs/fpAtlas-%06d-%d-%04d.fit run, rerun, run, camcol, field
psField the initial photometric calibration and point spread function fit by field http://das.sdss.org/imaging/%d/%d/objcs/%d/psField-%06d-%d-%04d.fit run, rerun, camcol, run, camcol, field

Spectroscopy

File Type Description DAS URL format format parameters
spPlate the 640 combined flux- and wavelength-calibrated spectra over all exposures (potentially spanning multiple nights) for a given mapped plate http://das.sdss.org/spectro/2d_%d/%04d/spPlate-%04d-%d.par rerun, plate, plate, mjd
spSpec line measurements and redshift determinations, as well as the spectrum, for a single object, summing over all of its exposures through a given mapped plate http://das.sdss.org/spectro/1d_%d/%d/1d/spSpec-%d-%04d-%03d.par rerun, plate, mjd, plate, fiber
sspp gzipped tar files Output of the SEGUE stellar parameters pipeline http://das.sdss.org/spectro/sspp_%d/%04d-%05d.tgz rerun, plate, mjd
spZline the emmission line fits from the Princeton-1D code, specBS http://das.sdss.org/spectro/2d_%d/%04d/spZline-%04d-%d.par rerun, plate, plate, mjd
spZall all spectroscopic classifications and redshifts from the Princeton-1D code, specBS http://das.sdss.org/spectro/2d_%d/%04d/spZall-%04d-%d.par rerun, plate, plate, mjd
spZbest spectroscopic classifications and redshifts from the Princeton-1D code, specBS http://das.sdss.org/spectro/2d_%d/%04d/spZbest-%04d-%d.par rerun, plate, plate, mjd
