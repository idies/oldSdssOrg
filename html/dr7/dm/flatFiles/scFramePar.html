List of Very Bright Star Frames

File Format: FTCL parameter file

Name: scFrame-rrrrrr-c.par where rrrrrr is the run number, and c is the column in the Imaging Array.

Produced by: ssc

Used by: ps

Size: < 10 Kb

Archived? No

Description

Contains a list of whole frames, cut out by ssc, which contain at least one very bright star.

Contents




version             'XXX'               # Version of software used. 
run                 XXX                 # Imaging run number. 
camCol              XXX                 # Column in the imaging camera. 
ssc_id              'XXX'               # SSC pipeline run identifier. 
ko_ver              'XXX'               # Version of known object catalog  
                                        # used. 



typedef struct {
    int       field;          # Field sequence number within the run.  0  
                              # indexed. 
    char      filter[20];     # Filter (u, g, r, i, or z). 
    int       row;            # Row center of star in trimmed image.  
                              # (pixels) 
    int       col;            # Column center of star in trimmed image.  
                              # (pixels) 
    float     mag;            # Catalog magnitude of star. 

} SCFRAME
