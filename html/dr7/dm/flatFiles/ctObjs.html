List of Targets in Tile Area

File Format: FITS binary table

Name: ctObjs.fit

Produced by: completeTilePre

Used by: completeTile

Size: Total of up to 10MB for a tiling run of 1000 sq degrees.

Archived? Yes

Description

List of all potential target objects within the area of sky tiled by a single tiling run. The primTarget and secTarget fields take the same value as the same fields in the tsObj files (see the description of that file for the legal values for those bit masks). If the object was targetted in the main survey then the targetting flags and priority are for the main survey; in this case the highest bit in the two flags will not be set. If the object was not targetted in the main survey then the target flags and priority are for the southern survey; in this case the highest bit in the two flags will be set.
