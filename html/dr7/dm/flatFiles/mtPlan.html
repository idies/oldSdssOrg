MT Frames Plan

File Format: FTCL parameter file

Name: mtPlan-ddddd.par, where ddddd is the int(MJD).

Produced by: mtframesPre

Used by: mtframes

Size: < 10 Kb

Archived? Yes

Description

Processing plan to control mtframes. Each plan targets one night's data.
