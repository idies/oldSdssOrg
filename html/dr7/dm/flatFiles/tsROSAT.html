ROSAT Catalog

File Format: FITS binary table

Name: tsROSAT.fit

Produced by: ROSAT folks

Used by: ts, target

Size:

Archived? Yes

Description

ROSAT source catalog. There is one file for the entire ROSAT catalog (as currently distributed by the ROSAT folks).

Primary Header


SIMPLE  =                    T
BITPIX  =                    8
NAXIS   =                    0
EXTEND  =                    T
VERSION = 'XXX     '           / ROSAT catalog version
END

Table Header

In the table below, the values in the flags field is a bit mask with the following enumerated types: 
typedef enum arRosatFlags
{
  AR_ROSAT_FLAG_NEARBY       =   0x1, // Nearby sources affecting SASS flux
                                      // determination.
  AR_ROSAT_FLAG_POSITION     =   0x2, // Possible problem with position
                                      // determination.
  AR_ROSAT_FLAG_EXTENDED     =   0x4, // Extended beyond SASS extraction radius.
  AR_ROSAT_FLAG_COMPLEX      =   0x8, // Complex diffuse emission pattern.
  AR_ROSAT_FLAG_MISSED       =  0x10, // Source missed by SASS.
  AR_ROSAT_FLAG_RECALCULATED =  0x20, // Source counts and extraction radius
                                      // recalculated.
  AR_ROSAT_FLAG_IMAGE        =  0x40, // Broad band image available.
  AR_ROSAT_FLAG_VARIABLE     =  0x80, // Variability flag.
  AR_ROSAT_FLAG_MBROAD       = 0x100, // Detected in broad band using
                                      // background map.
  AR_ROSAT_FLAG_LBROAD       = 0x200, // Detected in broad band using local
                                      // background.
  AR_ROSAT_FLAG_MHARD        = 0x400, // Detected in hard band using
                                      // background map.
  AR_ROSAT_FLAG_LHARD        = 0x800, // Detected in hard band using local
                                      // background.
  AR_ROSAT_FLAG_MSOFT        =0x1000, // Detected in soft band using
                                      // background map.
  AR_ROSAT_FLAG_LSOFT        =0x2000  // Detected in soft band using local
                                      // background.
};
