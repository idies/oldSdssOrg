Astrometric Standards Catalog

File Format: FITS binary table

Name: koTycho2-XX.fit for Tycho-2 catalogs, koUcac-XX.fit for UCAC catalogs, where XX is the stripe number.

Produced by: manually

Used by: astrom

Size: Big

Archived? No

Description

Input catalog of primary astrometric standards, one file per survey stripe, against which SDSS data is astrometrically calibrated by the astrom pipeline. Two different catalogs are used; 1) USNO CCD Astrograph Catalog (UCAC), the catalog of choice (but it doesn't currently cover the entire SDSS area), and 2) Tycho-2. The UCAC catalog files don't contain the proper motion, magnitude error, color, and flag fields.
