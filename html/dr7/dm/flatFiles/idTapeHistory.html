Tape History

File Format: FTCL parameter file

Name: idTapeHistory-llllll.par, where llllll is the tape label.

Produced by: iop

Used by: strategy (tape loading onto production system) Size: < 100 Kb.

Archived? Yes.

Description

Lists a summary of the contents of a single labelled imaging tape.

Contents

} TAPEHISTORY;
