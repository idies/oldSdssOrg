Tape Label Log

File Format: ASCII param file

Name: daTapeLog-yyyy-mm-id.par, where yyyy is the year, mm-id is the month followed by a id for multiple logs/month

Produced by: da, iop, telescope mountain operations, fnal tape operations

Used by: opdb

Size: Several 10's of Kb per file. At least one file per month, perhaps one per day/night.

Archived? Yes, in the opdb.

Description

A list of DLT tapes, their labels (VSNs) and what's on each tape.

Format

#
#
#	Log of tapes created and written at APO 1995 November
#
#

typedef struct {
	char label[20];		#VSN PP-AA-NN (no spaces, JE prefix for APO, JF for FNAL, JH for other)
	int  runLow;		# imaging/spectro run low id, 0=other
	int  runHi;		# imaging/spectro run hi id, 0=other
        int camCol;      # 1-6 = main camera, 7=gangs+logs, 8=mt, 9,10=spectrograph, 0=other
        char dateString[20];    # YYYY/MM/DD corresponding to UT midnite of night's observing
	int dup;		# 1 if duplicate tape 
	char qualityCheck[20];  # initials of person who ok'ed tape media
} TAPEID;
