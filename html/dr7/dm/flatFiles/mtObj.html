MT Objects

File Format: FITS binary table

Name: mtObj-ssssssss.fit, where ssssssss is the sequence number.

Produced by: mtframes

Used by: excal, kali

Size: 17.28 Kb per file (assuming < 49 stars detected). For a typical night (10 hours, with 10 sequences per hour), we'll have 100 such files, for a total of 1.728 Mb.

Archived? Yes

Description

Objects detected in the MT pipeline. Each file contains the object list for one MT sequence.

Primary Header




SIMPLE  =                    T
BITPIX  =                    8
NAXIS   =                    0
EXTEND  =                    T
VERSION = 'XXX     '           / Version of software used.
TELESCOP= 'XXX     '           / Telescope with which data was obtained.
MTFRA_ID= 'XXX     '           / Mtframes pipeline run identifier.
SEQID   =                  XXX / MT sequence number.

MJD     =                  XXX / MJD day number of night when data was obtained.
OBJNAME = 'XXX     '           / Name of field
FLAVOR  = 'XXX     '           / Flavor of this sequence (Pri, Sec)
MJDDATE =         XXXXX.XXXXXX / MJD at start of the first exposure
RA      =           XXX.XXXXXX / ra for last exposure in sequence (deg)
DEC     =           XXX.XXXXXX / dec for last exposure in sequence (deg)
EQUINOX =          XXXX.XXXXXX / equinox for last exposure in sequence (yr)
END

Frame Parameters Header

There is one HDU of this type. It contains some averaged photometric information (seeing and sky brightness) in each filter. There is one entry for each filter. Usually NAXIS2 will always be 5, however if a frame was not taken in a specific filter, or if its quality was bad, then there will not be an entry for that filter. 


XTENSION= 'BINTABLE'          
BITPIX  =                    8
NAXIS   =                    2
NAXIS1  =                   13
NAXIS2  =                  XXX
PCOUNT  =                    0
GCOUNT  =                    1
TFIELDS =                    4


TFORM1  = '1A      '          
TTYPE1  = 'filter  '           / Filter.
TFORM2  = '1E      '          
TTYPE2  = 'fwhm    '           / Median FWHM of brightest objects on the frame.
TUNIT2  = 'pixels  '

TUNIT3  = 'ADUs/pixel'        
TFORM4  = '1E      '          
TTYPE4  = 'skyErr  '           / Global sky value for the frame. (error)
TUNIT4  = 'ADUs/pixel'        

END

Object List Header

There is one HDU of this type. This contains the list of stars detected in this sequence. There is one entry for each star.
