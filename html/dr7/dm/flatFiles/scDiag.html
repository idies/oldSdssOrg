SSC Diagnostics

File Format: FTCL parameter file

Name: scDiag-rrrrrr-c.par where rrrrrr is the run number, and c is the column in the Imaging Array.

Produced by: ssc

Used by:

Size: < 10 Kb

Archived? Yes

Description

Quality analysis diagnostic output from a single ssc run.

Contents




version             'XXX'               # Version of software used. 
run                 XXX                 # Imaging run number. 
camCol              XXX                 # Column in the imaging camera. 
ssc_id              'XXX'               # SSC pipeline run identifier. 
ko_ver              'XXX'               # Version of known object catalog  
                                        # used. 



typedef struct {
    int       field;          # Field sequence number within the run.  0  
                              # indexed. 
    int       nWingStars;     # Number of wing stars. 
    int       nFrameStars;    # Number of full frame stars. 
    int       nLeading;       # Number of stars in leading astro chip. 
    int       nMatched;       # Number of stars matched in both astro  
                              # chips. 
    int       nGoodAstro;     # num valid stars matched in astro chips
    int       nFound;         # num valid stars found in photo chips
    int       nValid;         # Number of stars in both astro chips and  
                              # passed tests in photo chips. 
    float     dRow;           # Difference between row positions of stars  
                              # in leading vs trailing astro chips.  
                              # (pixels) 
    float     dCol;           # Difference between column positions of  
                              # stars in leading vs trailing astro chips.  
                              # (pixels) 
    int       lBias;          # Bias value in left-hand amp of camRow=1  
                              # chip. (scaled integers) 
    int       rBias;          # Bias value in right-hand amp of camRow=1  
                              # chip. (scaled integers) 
    int       median;         # Median of column 100 on camRow=1 chip.  
                              # (scaled integers) 
    float     median_gr;      # median g-r color for this field 
    float     median_ri;      # median r-i color for this field 

} SCDIAG;
