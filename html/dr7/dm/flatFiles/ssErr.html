simSurvey tracking errors

File Format: FTCL parameter file

Name: xxx.par, where base describes the errors.

Produced by: the user

Used by: proc ssErrorsGenerate

Size: < 10 Kb

Archived? no

Description

Simple model of tracking errors for the 2.5m telescope.

delNuInit and delMuInit are the initial offsets of the telescope pointing.

The following depend on time: ScaleX, ScaleY, theta (rotator angle), and phi (drift direction).

They depend on time with an offset, linear term, and two sine waves, each with an amplitude, wavelength, and phase (in degrees). The units of time for this dependence is frames.

The following parameters describe perfect tracking.
