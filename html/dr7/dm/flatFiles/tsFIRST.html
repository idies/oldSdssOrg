FIRST Catalog

File Format: FITS binary table

Name: tsFIRST.fit

Produced by: FIRST folks

Used by: ts, target

Size:

Archived? Yes

Description

FIRST source catalog. There is one file for the entire FIRST catalog (as currently distributed by the FIRST folks), with an HDU for every FIRST field on the sky.

Primary Header


SIMPLE  =                    T
BITPIX  =                    8
NAXIS   =                    0
EXTEND  =                    T
VERSION = 'XXX     '           / FIRST catalog version
END

Table Header

There is one of these HDUs per FIRST field, with roughly 20,000 fields over the entire sky and 40 sources per field.
