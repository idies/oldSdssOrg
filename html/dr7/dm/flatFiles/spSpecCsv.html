spSpec CSV

File Format: Comma Separated Values (CSV)

Name: spSpecCsv-mmmmm-pppp-fff.csv, where mmmmm is the mjd, pppp is the plate id number and fff is the fiber id number.

Produced by: DAS

Used by: end user

Archived? no

Description

This is a simple extraction from spSpec files. The csv headers are: 

logLambda,lambda,flux,fluxMinusContinuum,standardDeviation,mask
where
  • logLambda is calculated from COEFF0 and COEFF1 of the spSpec file
  • lambda (Angstrom) is 10 ^ logLambda
  • flux is first row of the primary header in the spSpec file
  • fluxMinusContinuum is second row of the primary header in the spSpec file
  • standardDeviation is the third row of the primary header in the spSpec file
  • mask is the fourth row of the primary header in the spSpec file
