Single-Exposure/Single-Camera Spectra

File Format: FITS image

Name: s-sc-eeeeeeee.fit, where s is the spectrograph (1,2), c is the camera band (b=blue,r=red), and eeeeeeee is the exposure number.

Produced by: spectro2d

Used by: spectro1d

Archived? Yes. Tracked file.

Description

An output spectrum from the 2-D spectroscopic pipeline for a single exposure for a single camera (blue or red).

Contains the spectra, one row per fiber, 320 fibers.
