Quasar Target Selection Locus Points

File Format: FTCL parameter file

Name: tsLocusPoints-XXX.par where XXX is a Galactic region identifier, one for the cap, N for the skirt

Produced by: quasar

Used by: target selection

Size: < 10 Kb

Archived? Yes

Description

Defines the locii used for quasar target selection.

Contents

The file contents must consist only of keywords (no tables). The actual contents are not controlled by the data model, as they are used only by the pipeline, as thus are controlled by the pipeline.
