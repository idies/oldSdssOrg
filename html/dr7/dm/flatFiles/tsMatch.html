List of matched objects

File Format: FITS binary table

Name: tsMatch-rrrrrr-c-v.fit, where rrrrrr is the imaging run number, c is the camera column (1-6) and v is the rerun number.

Produced by: ts

Used by: sx

Size: 2 * 2880 + (28 * Nmatches) {rounded up to nearest multiple of 2880). For a single pipeline run with 1000 fields, 400 objects per field, and typically 40 matches per field, 1126.08 Kb.

Archived? Yes, by sx.

Description

List of matches between objects, for export to SX. All matches with objects in this run are listed. No matches between other runs are given.
