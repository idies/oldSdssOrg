Combined and Merged Spectrum

File Format: FITS image

Name: spSpec-pppp-fff.fit, where pppp is the plugged-plate id number, and fff is the fiber id number,

Produced by: spectro1d

Used by:

Archived? Yes

Description

An output spectrum from the 1-D spectroscopic pipeline. It is the averaged spectrum over all exposures for a plugged plate (may span multiple sequences), with the separate spectrum from the blue and red cameras merged into a single spectrum.
