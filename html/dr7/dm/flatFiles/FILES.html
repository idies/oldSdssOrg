    Target Selection

    • ts
      • tsPre inputs
        • tsPlan -- processing plan
        • tsMatchParam -- Target Selection Matching Parameters
        • tsQuasarParam -- Quasar Target Selection Parameters
        • tsGalaxyParam -- Galaxy Target Selection Parameters
        • tsStarParam -- Star Target Selection Parameters
        • tsSerendipParam -- Serendipity Target Selection Parameters
        • tsStandardParam -- Standards Target Selection Parameters
        • tsLocusPoints -- Quasar Target Selection Locus Points
        • tsChunk -- list of segments in a resolved chunk of sky
        • tsObj -- limits -- calibrated objects
        • tsFIRST -- FIRST catalog
        • tsROSAT -- ROSAT catalog
      • additional inputs
      • outputs
        • tsObj -- limits -- calibrated objects
        • tsTargets -- spectroscopic targets selected from a chunk
    • export
      • outputs
        • Directory name
        • tsChunk (ts)
        • tsObj -- limits -- calibrated objects
        • tsField -- limits -- calibrated field info
        • tsTargets -- spectroscopic targets selected from a chunk
        • tsMasks -- culled and bad fields (optional)
    • tsManual
      • outputs
        • tsManual -- manually selected targets
    • tile
      • tilePre outputs
        • tiPlan -- processing plan
        • tiParam -- tunable parameters
        • tiInputTargets -- untiled targets
        • tiBoundariesIn -- targetting chunk boundaries
      • additional inputs
      • outputs
        • tiOutputTargets -- tiled and untiled targets
        • tiTileCenters -- tile centers
        • tiUnTiledCenters -- tile centers for plates which were never drilled or redesigned
        • tiBoundariesOut -- tiling chunk boundaries
        • tiBoundariesOut -- targetting chunk boundaries in new format for sql DB uploads
        • tilingMask -- targetting chunk mask in new format for sql DB uploads

        • ctObjs -- potential targets in tile area
      • additional inputs
        • tiTileCenters (tile)
      • outputs
        • plPlugMap -- final target list for tile

      • sopPre outputs
      • additional inputs
        • plPlugMapM -- (plug)
      • outputs
        • sdReport -- Spectro observing report
        • sdR -- Raw spectro data frames
        • idTapeSummary -- nightly log of what run/camcol is on which labeled tape
        • idTarTape (Tar tape logs)
        • opConfig (CCD readout configurations)
        • opECalib (CCD electronic calibrations)
        • opBC (CCD bad pixels)

    • spectro2d
      • spectro2dPre outputs
        • spPlan2d -- processing plan
        • spPlancomb -- combining plan
      • additional inputs
        • sdReport (specDA)
        • sdR (specDA)
        • plPlugMap (plug)
      • outputs
        • spPlate -- combined spectra over all exposures
    • spectro1d
      • spectro1dPre outputs
        • spPlan1d -- processing plan
        • spParam1d -- tunable parameters
      • additional inputs
        • spPlate (spectro2d)
        • plPlugMapM (plug)
      • outputs
        • spSpec -- limits -- line measurements and redshift determinations
        • spPlot -- gif and ps plots of the spectra
        • spDiag -- diagnostics
    • spectrodp
      • spectrodp inputs
        • tsObj -- the tsObj files
        • fpAtlas -- atlas images
        • spDiag -- spectro1d summary
        • plPlugMapM -- the plug mapping
        • spSpec -- headers of the spSpec file
      • outputs
        • spObj -- tsObj+spDiag+plPlugMapM files for each spectro object
        • spAtlas -- atlas images for each spectro object
        • stars-$label -- spObj for all objects spectro classed as stars
        • gals-$label -- spObj for all objects spectro classed as galaxies
        • qsos-$label -- spObj for all objects spectro classed as quasars
        • unk-$label -- spObj for all objects spectro classed as sky or unkown
        • main_gals-$label -- spObj for all objects spectro classed as galaxies and targeted as main galaxies
        • main_qsos-$label -- spObj for all objects spectro classed as quasars and targeted as main quasars
        • main_lrgs-$label -- spObj for all objects spectro classed as galaxies and targeted as LRG
