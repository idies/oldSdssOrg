Raw Spectroscopic CCD Frame

File Format: Non-standard FITS image (uses unsigned integers)

Name: sdR-cs-xxxxxxxx.fit, where c is the camera band (b=blue,r=red), s is the spectrograph (1 or 2), and xxxxxxxx is the running (ever increasing) exposure number. These may be kept in sync so frames from the red/blue side 1/2 instrument have the same exposure number.

Produced by: specDA

Used by: spectro2d

Size: 8.8 Mb per file.

Archived? Yes. Tracked file.

Description

A raw frame from the Spectroscopic Camera.
