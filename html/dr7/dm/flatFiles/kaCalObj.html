MT Calibrated Objects

File Format: FITS binary table

Name: kaCalObj-ssssssss.fit, where ssssssss is the sequence number.

Produced by: kali

Used by: ps, fcalib

Size: 8.64 Kb per file, assuming < 46 stars were detected.

Archived? Yes

Description

Calibrated objects detected in the MT pipeline. Each file contains the object list for one MT sequence.
