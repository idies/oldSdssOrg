Measured Spectroscopic Parameters

File Format: FTCL parameter file

Name: spSpecPar-pppp-dddd.par, where pppp is the plate id, and dddd the fiber id.

Produced by: spectro

Used by:

Size: < 10 Kb

Archived? Yes

Description

Redshift, line lists for each spectrum

Contents

plate   XXXX	#plate id (int)
mjd    XXXXX	#date plate observered

	float confidence;

	int nEmLines;
} SPPARAM;

	int fiberId;  		#must match plug map fiberId
