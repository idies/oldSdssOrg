FOOT User's Guide

Use the Footprint Server to create JPEG finder charts for a list of sky positions (RA, DEC in J2000).

Table of contents:

List of Coordinates

Provide a list of coordinates for which you want JPEG finder charts and/or GIF spectrograms.

The list can be either entered directly into the webform box or uploaded as a file from your machine. In either case, it must be a comma separated list containing fields for ra and dec. The order of these fields in the list is unimportant, but the first line of the list must contain the order of the fields in comma-separated format; e.g.: 

ra,dec

Valid formats (J2000):

  • ra in decimal degrees (D.ddddd)
  • dec in decimal degrees (+/-DD.ddddd)

    Output Content

    Indicate what content you want outputted by your query.

  • Imaging (target version) run/rerun/camcol/field/rowc/colc: create a JPEG finder chart image based upon the target version of the imaging data
  • Imaging (best version) run/rerun/camcol/field/rowc/colc: create a JPEG finder chart image based upon the best version of the imaging data
