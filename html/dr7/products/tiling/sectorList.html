sectorList files

sectorList is a parameter file which contains the information about each "sector" --- i.e., each region covered by a unique set of tiles. We used to refer to these as "overlap regions" (e.g. in Blanton et al. 2001 and Zehavi et al. 2001) but now have adopted the 2dF terminology. It has the structure: 

typedef struct {
   int index;
   int tileId[20];
   double area;
   int nTiles;
   long nTargets;
   long nRandom;
} TIOVERLAP;
These parameters are:
  • index: Index number of the sector
  • tileId[20]: List of tiles which overlap this sector
  • area: Area of the overlap region (square degrees) determined by Monte Carlo
  • nTiles: Number of tiles overlapping this sector
  • nTargets: Number of targets in this sector
  • nRandom: Number of Monte Carlo points in sector (to evaluate error in area)
