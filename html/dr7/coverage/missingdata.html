Missing Data

This page provides a list of small regions of data missing from the nominal coverage of each data release. The list is cumulative, i.e., missing DR1 data are still missing in DR5. Except where noted, even if some files are missing from the 'target' version of sky, the 'best' version of sky contains a complete, reprocessed version of the data.

NOTE: These tables also note missing fpFieldStat files. Although these are not used in the DAS, their absence is reported for completeness. These entries are noted in italics where applicable.
