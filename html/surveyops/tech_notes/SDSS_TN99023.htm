Other Systems

  • Telescope Servo System
    • Telescope azimuth model complete. Friction modelnearl complete for rotator. Will gather additional data during February trip.
    • "Fast Monitoring" analysis tool software development continues. Ready for testing at Telescope.
    • Data structures established for interface between MCP and interlocks system.
    • Initial stab at Visual Tcl GUI display for servo/interlocks status complete.

  • Interlock System
    • Working on controls for 15-degree stop.
    • Instrument change operator pendant installed and functional. MCP code modified to interface with control pendant. Will test MCP/pendant interface during week of February 23.
    • Telescope stow position switch andlight connected.
    • Telescope "safe-to-approach" light installed and connected.
    • Camera cart limit switches cabled and connected.
    • Instrument lift plate prox switches replaced (from sinking to sourcing to match other switch types.)
    • Instrument lift plate full-down position switch installed and wired.,

  • Slip Detection System
    Schematics completed. Board design and layout underway. Design work started on encoder mounting schemes.

  • Counterweights
    Design of new counterweight covers is complete. Drawings sent to APO. Plan for CVE to fabricate new covers.

  • Wind Baffle Altitude Brake
    Still working on solution.
    • Have found a replacement motor/brake assembly that reportedly will fit in existing envelope. Cost < $3K.
    • Sumitomo makes a right angle drive that may work. Still looking into geometry issues.

  • Fiber Mapper
    • All fiber mappe hardware installed and operational inthe APO support building.
    • Fully-loaded mapper hardware installed and operational in the APO support building.
    • Fiber mapper system tested by Pat Colestock and Denton Morris.
