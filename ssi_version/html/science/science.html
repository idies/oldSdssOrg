Data Release Schedule -- timeline for public release of SDSS data products.
  • December, 1998: SDSS Sampler CD-ROM, Volume 1 -- contains a sample of one of the first engineering runs of the SDSS telescope and imaging camera.
  • June, 2001: Early Data Release: 462 Square Degrees, 52,896 spectra. Check status of the SDSS EDR servers. The SDSS Moving Object Catalog.
  • April, 2003: Data Release One: 2099 Square Degrees, 186,240 spectra. Check status of the DR1 servers.
